Loading Vertex Buffers onto the Graphics Card

This textbox contains a bunch of numbers. The programmer has to let the graphics card know what these numbers mean. This program has been set up so that the numbers are in 3 groups of 6 numbers. Each groups defines one corner of the triangle. The number are in the format x, y, z, r, g, b where x, y and z are the position of the corner in 3D space, and r, g and b are the red, green and blue channels that define the colour of the point.

Try it yourself!

Play with the values and send them to the graphics card by clicking update!

For example, to move the top corner to the left change the first value to -1.0 

-1.0, 0.75, 0.0, 1.0, 0.0, 0.0,-0.75, -0.75, 0.0, 0.0, 1.0, 0.0, 0.75, -0.75, 0.0, 0.0, 0.0, 1.0

Or to change the bottom right corner to black change the last value to 0.0 

0.0, 0.75, 0.0, 1.0, 0.0, 0.0,-0.75, -0.75, 0.0, 0.0, 1.0, 0.0, 0.75, -0.75, 0.0, 0.0, 0.0, 0.0

See if you can predict what effect your changes will have.
